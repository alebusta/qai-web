4. O buscar: "Obtener ayuda" → "Hablar con un agente"

### 2. Twitter (OPCIÓN MÁS RÁPIDA)

**Cuentas**:
- **@GoogleHelp** - Soporte general Google
- **@GoogleAyuda** - Soporte en español
- **@GoogleOne** - Soporte específico Google One

**Cómo usar**:
1. Ir a Twitter
2. Enviar DM o tweet público mencionando @GoogleHelp
3. Mensaje corto: "Tengo problema con suscripción Google One asociada a perfil incorrecto. Necesito facturación empresarial. ¿Pueden ayudarme?"
4. Incluir Order number: SOP.3359-4751-7065-15633
5. Esperar respuesta (horas, no inmediato)

**Ejemplo tweet**:
```
@GoogleHelp Necesito ayuda: Suscripción Google One (Order: SOP.3359-4751-7065-15633) 
se asoció a perfil personal en vez de empresarial. Necesito facturación empresarial 
por razones fiscales. ¿Pueden ayudar? #GoogleOne #BillingIssue
```

### 3. Google One App (Chat en Vivo)

**Cómo acceder**:
1. Abrir app Google One en móvil
2. Tocar tu foto de perfil (esquina superior derecha)
3. Seleccionar "Ayuda"
4. Buscar "Contactar con soporte" o "Chat en vivo"
5. Si está disponible, usar chat

**Disponibilidad**: Horarios específicos (no 24/7)

## Información a Proporcionar

### Para Problemas de Facturación/Suscripciones

**Obligatorio**:
- Email asociado a la cuenta
- ID de perfil de pago (ej: 6405-1829-5336 para QAI)
- Número de orden o ID de suscripción
- Descripción clara del problema

**Recomendado**:
- Capturas de pantalla del problema
- Fechas relevantes
- Intentos previos de solución

### Ejemplo de Mensaje

```
Asunto: Suscripción Google One asociada a perfil incorrecto

Hola,

Tengo dos perfiles de pago en Google Payments:
1. Perfil personal: Alejandro Bustamante (ID: 4103-0888-6488)
2. Perfil empresarial: QAI (ID: 6405-1829-5336)

Mi suscripción a Google One se asoció automáticamente al perfil personal 
cuando intenté suscribirme con el perfil empresarial QAI.

Necesito:
- Asociar la suscripción al perfil QAI para facturación empresarial
- Configurar Tax ID empresarial (RUT: 78.313.539-6) para evitar VAT

¿Es posible transferir la suscripción o debo cancelar y resuscribir?

Gracias,
[Tu nombre]
```
