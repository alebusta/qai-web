# Mision QAI web refactoring

## Objetivo

Debes convertirte en un experto UX para crear una stunning página web corporativa de QAI, elegante, limpia, minimalita y clara que invite a los usuarios a querer conocer más de los que hacemos.

## Situación actual

Página web basica [qai.cl](http://qai.cl) al aire, sencilla, hecha en html con textos interesantes pero que no transmite lo que queremos transmitir como empresa. Ademas de muchos links de cosas inexistentes o sin contenido cread.

## Misión

Analizar detenidamente documento “Referentes Web Diseño IA Minimalista” que está en nuestro drive y tomalo como base para el rediseño de la página web. Para el contenido tomando en consideración nuestra filosofía, productos, servicios, estrategia de negocio (ver manifiesto y otros documentos en /Empresa/01_estrategia). Considera también el minimalismo radical, si hay cosas que no tenemos no las pongamos, un delicado equilbrio para que no se vea despoblada pero tampoco inventado o magnificando cosas. Considera dejar planificadas secciones que aun no vamos a tener pero que tienen que tener room como demos o blogposts. Puedes también navegar en [qai.cl](http://qai.cl) para que veas el contenido y estimes qué cosas reutilizar.
